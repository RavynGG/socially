# Socially

Open source social media platform built with Next.js, TypeScript & Tailwind CSS.

## Features

- User profiles with bios, avatars, usernames, and settings
- Text posts with timestamps and user attribution
- Like/unlike functionality
- Follow/unfollow system
- Feed showing posts from followed users
- Protected API routes with server-side authorization
- Responsive UI using React and Next.js App Router
- Prisma ORM for database management
- Email service for verification and password resets
- Environment-based configuration
- NextAuth v4 authentication (sign-up, login, email verification, password reset)
